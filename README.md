# scss-compile

## About

Simple tool to compile scss to css with gulp watch; includes the following mixins:

* Bourbon
* Neat

## Install

* Clone repo
* ```bash install.sh```

## Usage

* Add scss partials to ```scss/partials/```
* Include partials in ```scss/app.scss```
* From command line use ```gulp``` or ```gulp watch```
